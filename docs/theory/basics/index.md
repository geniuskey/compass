This chapter is the non-mathematical entry point for the theory section. It explains the physical story in plain language before the optics, sensor, and simulation chapters introduce equations and implementation details.

## Boundary with the deeper chapters

- **Basics** answers "what is happening?" without requiring equations.
- **Optics** answers "which wave-optics law explains it?"
- **Image Sensor** answers "where does it happen in the pixel stack?"
- **Simulation** answers "how does COMPASS compute it?"

If two pages seem to cover the same topic, read the Basics page for intuition and the later chapter page for the precise model.
