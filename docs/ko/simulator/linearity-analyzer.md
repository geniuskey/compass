# 선형성 분석기

조절 가능한 비선형성과 니 포인트로 센서 전달 곡선 선형성을 분석합니다. 전달 함수와 잔차 플롯을 나란히 표시합니다.

::: info 모델 범위
이 브라우저 도구는 직관 형성, 상대 경향 비교, 설계 공간 탐색에 사용하세요. 로컬에서 실행되는 간이 모델이며 RCWA/FDTD sign-off, 실리콘 보정, 벤더 공정 데이터를 대체하지 않습니다.
:::

## 선형성 모델

이상적 전달 함수는 선형입니다:

**DN_ideal = (Signal / FWC) × DN_max**

실제 센서는 다음 원인으로 선형성에서 벗어납니다:
- 전하-전압 변환 비선형성 (소스 팔로워)
- 전압에 따른 커패시턴스 변동 (접합 커패시턴스)
- ADC 적분 비선형성 (INL)

### 비선형성 지표

**NL(%) = max|DN_actual - DN_ideal| / DN_max × 100**

### 니 포인트

니 포인트는 비선형성이 본격적으로 시작되는 지점을 정의합니다. 니 포인트 이하에서는 응답이 거의 선형이며, 이상에서는 압축 또는 확장 효과가 증가합니다.

### 주요 지표
- **최대 비선형성 %** — 이상적 응답에서 최대 편차
- **선형 범위 %** — ±1% NL 이내인 풀 스케일 비율
- **RMS 오차 %** — 전체 범위 RMS 편차

::: warning
높은 비선형성(>2%)은 색 정확도, HDR 합성 품질, 측광 측정을 저하시킵니다. 대부분의 머신 비전 응용은 NL < 1%를 요구합니다.
:::
