# 광자 전달 곡선 (PTC)

노이즈-신호 관계를 로그-로그 플롯으로 시각화하여 읽기 노이즈, 변환 이득, 풀 웰 용량, PRNU를 통합 분석합니다.

<PhotonTransferCurve />

## 노이즈 모델

PTC는 평균 신호 대비 노이즈 표준편차를 로그-로그 스케일로 표시합니다. 세 가지 노이즈 영역이 구분됩니다:

### 읽기 노이즈 영역 (기울기 0)
저신호 구간에서 읽기 회로의 고정 노이즈가 지배합니다.

### 샷 노이즈 영역 (기울기 0.5)
포톤 샷 노이즈(포아송 통계)가 지배하는 구간으로, sigma = sqrt(N)이므로 기울기가 0.5입니다. 변환 이득은 이 영역에서 추출합니다.

### PRNU 영역 (기울기 1.0)
높은 신호 레벨에서 픽셀 간 이득 편차(PRNU)가 지배하며, sigma = PRNU x N이므로 기울기가 1.0입니다.

**sigma^2 = sigma_read^2 + N + (PRNU x N)^2**

::: tip
영역 간 교차점은 핵심 진단 파라미터입니다. Shot/Read 교차점이 높으면 읽기 노이즈가 과다하고, PRNU/Shot 교차점이 낮으면 제조 균일성이 불량합니다.
:::
