# PRNU / DSNU 시각화

이미지 센서의 고정 패턴 노이즈(FPN)를 시각화합니다 — 광응답 비균일성(PRNU)과 암신호 비균일성(DSNU). 공간 노이즈 패턴과 통계 분포를 관찰합니다.

## 고정 패턴 노이즈

고정 패턴 노이즈(FPN)는 프레임 간 일정한 픽셀별 변동입니다. 두 성분으로 구성됩니다:

### PRNU (광응답 비균일성)
신호 레벨에 비례하는 픽셀 간 이득 변동:

**σ_PRNU = PRNU_factor × Signal**

픽셀 기하, 마이크로렌즈 정렬, 포토다이오드 도핑 차이로 발생합니다. 현대 센서에서 보통 0.5–2%.

### DSNU (암신호 비균일성)
신호와 무관한 픽셀 간 오프셋 변동:

**σ_DSNU = DSNU [DN RMS]**

결정 결함과 계면 트랩에 의한 암전류 변동으로 발생합니다.

### 전체 FPN

**σ_FPN = sqrt(σ_PRNU² + σ_DSNU²)**

::: tip
PRNU는 높은 신호 레벨에서, DSNU는 어두운 영역에서 지배적입니다. FPN은 픽셀별 보정(플랫 필드 및 다크 프레임 차감)으로 교정할 수 있습니다.
:::
