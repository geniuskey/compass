# 마이크로렌즈 광선 추적

초타원체(superellipsoid) 마이크로렌즈를 통한 2D 광선 추적 시뮬레이션입니다. 렌즈 형상과 입사 조건을 변경하며 집광 효율을 분석하세요.

<MicrolensRayTrace />

::: info 모델 범위
이 브라우저 도구는 직관 형성, 상대 경향 비교, 설계 공간 탐색에 사용하세요. 로컬에서 실행되는 간이 모델이며 RCWA/FDTD sign-off, 실리콘 보정, 벤더 공정 데이터를 대체하지 않습니다.
:::

## 시뮬레이션 원리

스넬의 법칙(n₁ sin θ₁ = n₂ sin θ₂)을 적용하여 마이크로렌즈 표면에서의 굴절을 계산합니다. 굴절된 광선이 포토다이오드 영역에 도달하는 비율로 집광 효율을 평가합니다.

### 주요 파라미터
- **렌즈 높이/반경**: 곡률을 결정하며, 초점 거리에 직접 영향
- **형상 파라미터 (p)**: p=2는 구형, p>2는 더 평평한 상단 프로파일
- **CRA 각도**: 주광선각이 클수록 광선이 인접 픽셀로 누설 (크로스토크 증가)

::: info
기하 광학 근사입니다. 파장 규모의 회절 효과는 RCWA/FDTD 시뮬레이션을 참조하세요.
:::
