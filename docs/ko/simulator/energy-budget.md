# 에너지 버짓 분석기

픽셀 스택을 통과하면서 모든 광자의 에너지가 어디로 가는지 추적합니다. 각 파장에서 입사 에너지는 반사, 각 레이어에서의 흡수, 투과로 나뉩니다.

## 에너지 버짓 이해하기

기본 제약 조건은 에너지 보존입니다: **R + A + T = 1**

- **반사(R)** — 계면에서 반사되는 빛, 특히 공기-마이크로렌즈 및 BARL-실리콘 경계에서
- **실리콘 흡수** — 유효 신호; 양자 효율과 같습니다
- **컬러 필터 흡수** — 의도적인 파장 선택적 흡수
- **BARL/평탄화/마이크로렌즈 흡수** — 기생 손실 (이상적으로 0)
- **투과(T)** — 실리콘을 통과하여 흡수되지 않은 빛, 얇은 실리콘에서 적색/NIR 파장에서 유의미

::: info
파란 광자(λ ≈ 450nm)는 실리콘의 처음 0.5μm 내에서 흡수되는 반면, 빨간 광자(λ ≈ 650nm)는 완전한 흡수를 위해 2-3μm가 필요합니다.
:::
