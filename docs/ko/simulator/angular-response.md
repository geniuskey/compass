# 각도 응답 시뮬레이터

실제 카메라에서 빛은 센서 위의 픽셀 위치와 렌즈 설계에 따라 다양한 각도로 픽셀에 도달합니다. 주광선각(CRA)은 센서 가장자리에서 20-30°에 달할 수 있으며, QE에 상당한 영향을 미칩니다.

## CRA 효과

입사각이 증가하면:
- **반사율 증가** — 프레넬 효과로 인해 (특히 s-편광에서)
- **유효 경로 길이 증가** — 각 레이어를 통과하는 경로가 길어져 박막 간섭 조건이 변화
- **QE 일반적으로 감소** — 감소율은 BARL 설계와 컬러 필터에 따라 달라짐

::: tip
잘 설계된 BARL은 15° CRA까지 수직 입사 QE의 80% 이상을 유지합니다. 20° 이상에서는 대부분의 픽셀 설계에서 QE 저하가 상당해집니다.
:::
