# 색 정확도 분석기

컬러 필터의 분광 감도로부터 색 재현 정확도를 평가합니다. 색 보정 행렬(CCM)을 적용하고 ΔE*ab 지표로 색차를 측정하세요.

<ColorAccuracyAnalyzer />

## 색 재현 파이프라인

1. **분광 감도 측정**: RGB 채널별 파장 응답 함수
2. **색 보정 행렬(CCM)**: 센서 RGB → sRGB 선형 변환 (3×3 행렬)
3. **ΔE\*ab 계산**: CIE L\*a\*b\* 색공간에서의 유클리드 색차

### ColorChecker 평가
X-Rite ColorChecker 24패치를 사용한 표준 평가 절차:
- D65 광원(주광) 하에서 각 패치의 센서 응답 계산
- CCM 적용 후 기준값과 비교
- ΔE\*ab < 3이면 육안으로 구별 불가, < 1이면 정밀 색 재현

### 필터 설계 영향
- 협대역 필터: 색 분리 우수하지만 감도 저하
- 광대역 필터: 감도 높지만 색 순도 감소 및 CCM 보정 부담 증가

::: info
이상적인 센서는 Luther 조건(분광 감도 ∝ 등색 함수의 선형 결합)을 만족해야 합니다.
:::
