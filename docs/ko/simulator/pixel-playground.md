# 픽셀 설계 놀이터

종합 픽셀 설계 도구입니다. BSI CMOS 픽셀의 모든 파라미터 — 실리콘 두께부터 BARL 서브레이어까지 — 를 구성하고 QE, 반사율, 에너지 버짓, 스택 구조에 대한 복합 효과를 확인하세요.

<PixelDesignPlayground />

## 설계 가이드라인

### QE 최대화
1. **두꺼운 실리콘** (3-4 μm) — 적색/NIR 흡수 향상
2. **최적화된 BARL** — 계면 반사 최소화
3. **얇은 컬러 필터** — 통과대역 외부의 기생 흡수 감소
4. **저손실 평탄화** 레이어

### 크로스토크 최소화
1. 컬러 필터 흡수를 높게 유지 (날카로운 스펙트럼 에지를 가진 두꺼운 CF)
2. 실리콘 투과광 감소 (장파장에서 얇은 Si에 해당)

::: tip
프리셋으로 시작하여 한 번에 하나의 파라미터를 조정하면서 효과를 이해하세요. "BSI 1μm"과 "High QE" 프리셋을 비교하여 실리콘 두께와 BARL 최적화의 영향을 확인하세요.
:::
