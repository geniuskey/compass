# 픽셀 스케일링 트렌드

픽셀 피치 축소에 따른 핵심 성능 지표의 변화를 시각화합니다. 물리적 한계와 상용 센서 데이터를 비교하세요.

<PixelScalingTrends />

## 스케일링 법칙

| 지표 | 스케일링 | 설명 |
|------|---------|------|
| FWC (풀웰 용량) | ∝ pitch² | 포토다이오드 면적에 비례 |
| 나이퀴스트 주파수 | ∝ 1/pitch | 작은 픽셀 = 높은 해상도 |
| SNR (최대) | ∝ pitch | √FWC에 비례 |
| 회절 제한 QE | 에어리 디스크 vs 픽셀 면적 | 작은 픽셀에서 급격히 감소 |

### 핵심 트레이드오프
- **해상도 vs 감도**: 피치 축소는 해상도를 높이지만 광자 수집 면적이 줄어 SNR 저하
- **회절 한계**: 픽셀이 에어리 디스크보다 작아지면 인접 픽셀로 에너지 누설
- **실용적 하한**: ~0.5μm 이하에서는 회절과 크로스토크가 심각하게 성능 제한

::: info
참조 데이터는 Sony IMX 시리즈 등 상용 센서 사양을 기반으로 합니다.
:::
