# FDTI / BDTI 픽셀 시뮬레이터

가상의 Deep Trench Isolation CMOS 이미지 센서 픽셀을 만들고 FDTI/BDTI 선택, 트렌치 폭, BDTI 깊이, 라이너 재료, 파장, 주광선 각도가 광학장 구속에 미치는 영향을 확인합니다.

## 모델

이 시뮬레이터는 브라우저에서 빠르게 동작하는 근축/광선 근사 모델입니다. 목적은 설계 리뷰와 시각적 QA입니다.

- 왼쪽 패널은 마이크로렌즈, 컬러 필터, 전면 산화막/게이트 스택, 실리콘, 포토다이오드, DTI 트렌치를 포함한 3픽셀 단면을 보여줍니다.
- FDTI는 실리콘 전체 두께를 관통하는 트렌치로 모델링합니다. BDTI는 후면 광 입사면에서 실리콘 내부로 내려가는 트렌치로 모델링하며 깊이를 선택할 수 있습니다.
- 오른쪽 패널은 포토다이오드 수집 영역이 겹쳐진 근사 광학 강도 맵을 렌더링합니다.
- 시각적 QA 시나리오는 현재 설계를 DTI 없음, 얕은 BDTI, 관통 FDTI+경사 입사 조건과 비교합니다.

사인오프 수준 정확도는 COMPASS의 RCWA 또는 FDTD 워크플로에서 같은 구조를 검증하고, 이 뷰는 빠른 사전 점검용으로 사용하는 것이 좋습니다.
