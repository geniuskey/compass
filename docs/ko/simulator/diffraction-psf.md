# 회절 PSF 뷰어

## 에어리 패턴

원형 개구를 통과한 빛의 회절 패턴은 베셀 함수로 기술됩니다:

**PSF(x) = [2J₁(x) / x]²**

여기서 x = π·D·sin(θ)/λ, D는 개구 직경입니다.

### 에어리 디스크
- 첫 번째 영점까지의 반경: r = 1.22·λ·F#
- 중심 디스크에 전체 에너지의 약 84% 집중
- 나머지 에너지는 동심원 링 패턴으로 분포

### 픽셀과의 관계
- **에어리 디스크 < 픽셀**: 대부분의 에너지가 단일 픽셀에 수집 (해상도 제한 = 픽셀)
- **에어리 디스크 > 픽셀**: 에너지가 인접 픽셀로 분산 (해상도 제한 = 회절)
- f-넘버가 작을수록(밝은 렌즈) 에어리 디스크가 작아져 집광에 유리

::: tip
F# = 2.8, λ = 550nm일 때 에어리 디스크 직경은 약 3.8μm로, 1μm 피치 픽셀에서는 다수 픽셀에 걸칩니다.
:::
