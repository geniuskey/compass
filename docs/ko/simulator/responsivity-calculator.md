# 응답도 계산기

TMM 기반 양자 효율에서 분광 응답도를 계산합니다. 채널별 응답도를 이상적 실리콘 포토다이오드 응답과 비교합니다.

<ResponsivityCalculator />

## 응답도

분광 응답도는 QE를 전기적 출력 지표로 변환합니다:

**R(λ) = QE(λ) × qλ / (hc) = QE(λ) × λ_nm / 1240 [A/W]**

여기서:
- **q** — 전자 전하 (1.602 × 10⁻¹⁹ C)
- **h** — 플랑크 상수 (6.626 × 10⁻³⁴ J·s)
- **c** — 광속 (2.998 × 10⁸ m/s)
- **λ_nm** — 나노미터 단위 파장

### 채널별 응답

각 색 채널(R, G, B)은 다음에 의해 형성된 고유한 응답도 곡선을 가집니다:
- 컬러 필터 투과 스펙트럼
- 각 파장에서의 실리콘 흡수 깊이
- 반사 방지 코팅 효율
- 마이크로렌즈 집광 효율

### 이상적 Si 포토다이오드

이상적 응답은 모든 파장에서 QE = 1로 가정합니다:

**R_ideal(λ) = λ_nm / 1240 [A/W]**

::: tip
피크 응답도 파장은 피크 QE 파장과 다릅니다. 응답도에 λ/1240 인자가 포함되어 피크가 장파장 쪽으로 이동하기 때문입니다.
:::
