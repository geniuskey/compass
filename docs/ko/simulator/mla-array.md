# MLA 어레이 시각화

마이크로 렌즈 어레이(MLA)의 형상을 배열 패턴, 비대칭 렌즈 반경, 곡률 파라미터를 조절하며 시각화합니다. 등고선, 등비율 단면, 3D 와이어프레임, 2D 광선 추적 뷰를 전환할 수 있습니다.

## 시뮬레이션 원리

각 마이크로렌즈는 X/Y 방향 독립적인 반경을 가진 초타원(superellipse) 프로파일로 모델링됩니다.

**z(r) = h (1 - r^2)^{1/(2alpha)}**

정규화된 반경 거리 r은 초타원 방정식으로 계산됩니다:

**r = (|x/Rx|^n + |y/Ry|^n)^{1/n}**

### 주요 파라미터

- **Rx, Ry** — 초타원 풋프린트의 반축. Rx = Ry이면 원형 대칭, 다르면 비등방성(anamorphic) 렌즈
- **형상 파라미터 n** — 경계 형상 조절: n=2는 타원, n>2는 더 직사각형에 가까운 풋프린트
- **곡률 alpha** — 새그 프로파일의 급경사도 조절. alpha=1은 표준 포물선형, alpha<1은 평평한 상단, alpha>1은 뾰족한 정점
- **간격** — 배열에서 렌즈 중심 간 피치. 충전률은 렌즈 면적 대 셀 면적의 비율

### 뷰 모드

- **등고선** — 렌즈 풋프린트 형상을 등고선으로 표시하는 탑다운 뷰
- **단면** — x/y 등비율 스케일링으로 XZ, YZ, 대각선 프로파일 비교
- **3D 표면** — 앙각/방위각 조절 가능한 와이어프레임 렌더링
- **광선 추적** — XZ 평면에서 스넬의 법칙 기반 2D 기하 광학 광선 추적

::: info
이 시뮬레이터는 MLA 형상과 배열 패턴에 초점을 맞춥니다. 전체 픽셀 스택(실리콘, BARL, 컬러 필터, 평탄화)을 포함한 단일 픽셀 광선 추적은 [마이크로렌즈 광선 추적](./microlens-raytrace)을 참조하세요.
