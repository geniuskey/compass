# TMM QE 계산기

Transfer Matrix Method를 사용하여 BSI CMOS 픽셀의 양자 효율 스펙트럼을 계산합니다. 실리콘 두께, BARL 레이어, 입사각을 조정하여 실시간 결과를 확인하세요.

<TmmQeCalculator />

## 작동 원리

TMM은 각 레이어를 복소 굴절률을 가진 박막으로 모델링합니다. 2×2 전달 행렬을 통해 스택을 통과하는 빛의 전파를 계산하여 반사율(R), 투과율(T), 레이어별 흡수율(A)을 구합니다. QE는 실리콘 포토다이오드 레이어에서 흡수되는 빛의 비율과 같습니다.

### 주요 파라미터
- **실리콘 두께**는 NIR 감도를 제어합니다 — 두꺼운 Si는 장파장 광자를 더 많이 흡수합니다
- **입사각**은 카메라 렌즈로부터의 주광선각(CRA) 효과를 시뮬레이션합니다
- **편광** — 실제 장면은 비편광입니다; s/p 모드는 개별 성분을 보여줍니다

::: info 물리 참고
이것은 1D 평면 근사입니다 — 마이크로렌즈 집광, DTI 산란, 횡방향 크로스토크는 고려하지 않습니다. 이러한 효과를 위해서는 전체 RCWA/FDTD 솔버를 사용하세요.
:::
