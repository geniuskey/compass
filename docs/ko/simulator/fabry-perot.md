# 파브리-페로 시각화

단층 박막에서의 다중 반사 간섭을 시각화합니다. 프레넬 계수와 페이저 다이어그램을 통해 박막 광학의 기초를 이해하세요.

<FabryPerotVisualizer />

## 간섭 원리

박막의 상하 경계면에서 반사된 빛이 간섭합니다. 위상차는 다음과 같습니다:

**δ = 4π·n·d·cos(θ)/λ**

여기서 n은 박막 굴절률, d는 두께, θ는 박막 내 굴절각입니다.

### 쿼터파 조건
반사율을 최소화하려면 박막 두께가 λ/(4n)의 홀수 배여야 합니다. 이때 두 반사파가 소멸 간섭하여 반사가 억제됩니다.

### 페이저 다이어그램
- 각 반사파를 복소 평면의 벡터(페이저)로 표현
- 벡터의 합이 전체 반사 진폭을 결정
- 다중 반사의 기여가 점차 감소하는 과정을 시각적으로 확인

::: tip
이미지 센서의 BARL(반사 방지 레이어)은 이 원리를 다층으로 확장하여 넓은 파장 범위에서 반사를 줄입니다.
:::
