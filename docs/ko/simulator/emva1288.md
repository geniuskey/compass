# EMVA 1288 대시보드

EMVA 1288 표준(Release 4.0) 기반 통합 센서 특성화 대시보드입니다. 픽셀 설계 파라미터로부터 주요 성능 지표를 시뮬레이션합니다.

## EMVA 1288 표준

유럽 머신 비전 협회(EMVA) 1288 표준은 이미지 센서 성능 측정 및 보고를 위한 통합 방법론을 정의합니다.

### 감도 지표
- **양자 효율 (QE)** — 광자-전자 변환 효율
- **응답도** — 단위 조도당 출력 신호 (A/W)
- **절대 감도 임계값** — 최소 검출 가능 광자 수

### 노이즈 지표
- **시간적 암노이즈** — 읽기 노이즈 (e- RMS)
- **PRNU** — 픽셀 간 이득 변동
- **SNR_max** — 최대 달성 가능 SNR = sqrt(FWC)

### 용량 지표
- **포화 용량 (FWC)** — 픽셀당 최대 전하
- **다이나믹 레인지** — FWC 대 노이즈 플로어 비율
- **유효 비트** — 노이즈를 고려한 실효 ADC 해상도

### 등급 시스템
파라미터는 품질에 따라 색상 코딩됩니다:
- **녹색** — 해당 지표에서 우수한 성능
- **파란색** — 일반적/수용 가능한 성능
- **빨간색** — 일반 수준 이하, 응용 적합성 제한 가능

::: tip
SNR 곡선은 실제 vs 이상적(샷 노이즈 제한) 성능을 보여줍니다. 곡선 간 차이는 센서가 이상적 광자 계수에서 얼마나 벗어나는지를 나타냅니다.
:::
