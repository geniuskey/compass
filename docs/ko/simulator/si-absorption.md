# 실리콘 흡수 깊이

파장별 실리콘 흡수계수와 광 침투 깊이를 시각화합니다. Beer-Lambert 법칙을 기반으로 실리콘 두께에 따른 광자 흡수 프로파일을 확인하세요.

<SiliconAbsorptionDepth />

## 물리 배경

빛이 실리콘에 입사하면 Beer-Lambert 법칙에 따라 지수적으로 감쇠합니다:

**I(z) = I₀ · exp(-αz)**

여기서 흡수계수 α = 4πk/λ (k는 소광계수, λ는 파장)입니다.

### 파장별 특성
- **청색광 (400-500nm)**: α가 크므로 표면 근처 ~0.1-1μm에서 대부분 흡수
- **녹색광 (500-600nm)**: 침투 깊이 ~1-3μm
- **적색/NIR (600-1000nm)**: α가 작아 깊이 침투 — 두꺼운 실리콘이 필요

::: tip
BSI 센서에서 실리콘 두께를 3μm 이상으로 늘리면 적색 QE가 크게 향상되지만, 광학적 크로스토크가 증가할 수 있습니다.
:::
