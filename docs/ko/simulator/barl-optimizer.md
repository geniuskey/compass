# 박막 스택 설계기

레이어 수, 순서, 재질, 두께를 자유롭게 조절하여 다층 박막 코팅을 설계합니다. 반사율과 투과율 스펙트럼을 실시간으로 확인할 수 있습니다.

<BarlOptimizer />

## 지원 재질 (20+)

| 분류 | 재질 |
|------|------|
| 저굴절률 유전체 | SiO₂, MgF₂, CaF₂, Al₂O₃ |
| 고굴절률 유전체 | Si₃N₄, HfO₂, TiO₂, Ta₂O₅, ZrO₂, Nb₂O₅, AlN |
| 반도체 | Silicon, Ge, SiC, ZnS, ZnSe |
| 도체 / TCO | Tungsten, ITO |
| 폴리머 / 유리 | Polymer (n=1.56), BK7 Glass |

## 내장 프리셋

- **BARL 4층**: 기본 BSI 픽셀 반사 방지 스택 (SiO₂/HfO₂/SiO₂/Si₃N₄)
- **2층 AR**: 간단한 MgF₂/TiO₂ 반사 방지 코팅
- **광대역 AR**: 넓은 스펙트럼 커버리지를 위한 4층 설계
- **HR 블루 미러**: 청색 파장 고반사율 6층 스택
- **NIR 차단 필터**: 유리 기판 위 5층 근적외선 차단 필터

::: tip
"자동 최적화" 버튼으로 목표 파장 대역에서 평균 반사율을 최소화하는 두께 조합을 찾으세요. 최적화 전에 레이어를 자유롭게 추가, 제거, 재배치할 수 있습니다.
:::
