# 컬러 필터 설계

RGB 컬러 필터 스펙트럼 응답을 설계하고 CIE 1931 색도 다이어그램에서 결과 색재현 범위를 시각화합니다. 중심 파장, 대역폭, 피크 투과율을 조정하여 색 재현을 최적화하세요.

::: info 모델 범위
이 브라우저 도구는 직관 형성, 상대 경향 비교, 설계 공간 탐색에 사용하세요. 로컬에서 실행되는 간이 모델이며 RCWA/FDTD sign-off, 실리콘 보정, 벤더 공정 데이터를 대체하지 않습니다.
:::

## 컬러 필터 설계 트레이드오프

- **좁은 대역폭** → 순수한 색상, 넓은 가멋, 하지만 낮은 QE (통과하는 빛이 적음)
- **넓은 대역폭** → 높은 QE, 하지만 채널 간 스펙트럼 겹침 증가 (크로스토크)
- **중심 파장** 배치는 sRGB 대비 가멋 형태와 커버리지를 결정

::: info
참조용으로 sRGB 가멋 삼각형이 표시됩니다. 실제 이미지 센서는 컬러 필터 설계에 따라 일반적으로 sRGB 가멋 면적의 70-90%를 달성합니다.
:::
