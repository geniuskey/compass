# 픽셀 SNR vs 조도

광자 수(조도)에 따른 신호 대 잡음비를 표시합니다. 소스별 노이즈 분해를 시각화하고 실제 센서 성능과 이상적 샷 노이즈 한계를 비교합니다.

## SNR 모델

**SNR = N / sqrt(σ_read² + N + (PRNU × N)²)**

여기서:
- **N** — 신호 전자 (광자 수 × QE)
- **σ_read** — 읽기 노이즈 (전자 RMS)
- **PRNU** — 광응답 비균일성 계수

### 노이즈 영역

| 영역 | 지배적 노이즈 | SNR 기울기 |
|------|-------------|-----------|
| 저조도 | 읽기 노이즈 | SNR ∝ N |
| 중간 범위 | 샷 노이즈 | SNR ∝ √N |
| 고신호 | PRNU | SNR 포화 |

### 주요 마커
- **Unity SNR (0 dB)** — 신호가 노이즈와 같은 지점, 최소 유효 노출
- **20 dB 임계값** — 일반적으로 수용 가능한 최소 화질
- **포화** — 풀 웰 용량 도달

::: tip
실제와 이상적(샷 노이즈 제한) SNR 곡선 사이의 차이는 센서 전자 회로가 성능을 얼마나 저하시키는지 보여줍니다. 차이가 작을수록 더 나은 센서 설계입니다.
:::
