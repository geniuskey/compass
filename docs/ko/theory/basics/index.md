## 심화 챕터와의 경계

- **입문**은 수식 없이 "무슨 일이 일어나는가?"에 답합니다.
- **광학**은 "어떤 파동 광학 법칙으로 설명되는가?"에 답합니다.
- **이미지 센서**는 "픽셀 스택의 어느 위치에서 일어나는가?"에 답합니다.
- **시뮬레이션**은 "COMPASS가 그것을 어떻게 계산하는가?"에 답합니다.

두 페이지가 같은 주제를 다루는 것처럼 보이면, 입문 페이지는 직관을 만들기 위해 읽고 뒤쪽 챕터 페이지는 정확한 모델을 확인하기 위해 읽으면 됩니다.
